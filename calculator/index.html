Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Dimana :

L = Luas

a = panjang alas

t = tinggi





Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = a + b + c

Dimana :

K = Keliling

a = sisi a

b = sisi b

c = sisi c






Luas Jajargenjang

Rumus Luas Jajargenjang yaitu :

L = a x t

Dimana :

L = Luas

a = panjang alas

t = tinggi





Keliling Jajargenjang

Rumus Keliling Jajargenjang yaitu :

K = 2 x (a + b)

Dimana :

K = Keliling

a = sisi a

b = sisi b
